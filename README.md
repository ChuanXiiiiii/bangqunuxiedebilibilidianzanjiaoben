# B站自动点赞脚本

> 使用 Python + Playwright 实现对 B站首页视频的自动观看与点赞。

---

## 项目结构

```
bilibili/
├── main.py          # 脚本入口，运行这个文件
├── liker.py         # 核心逻辑（登录检测 / 链接收集 / 点赞）
├── config.py        # 所有可调参数
├── requirements.txt # Python 依赖
├── setup.sh         # 一键安装脚本
└── logs/
    └── liked_videos.log  # 运行日志（自动生成）
```

---

## 快速开始

### 第一步：安装依赖（只需执行一次）

```bash
bash setup.sh
```

### 第二步：运行脚本

```bash
python main.py
```

### 第三步：在浏览器中手动登录

脚本启动后会弹出 Chromium 浏览器并打开 B站首页，你需要：

1. 点击页面右上角的 **登录** 按钮
2. 使用扫码/账号密码完成登录
3. 登录成功后，脚本**自动继续** —— 无需任何操作

---

## 运行流程

```
启动脚本
  ↓
打开浏览器 → B站首页
  ↓
等待你手动登录（最长 180 秒）
  ↓
滚动首页，收集 20 个视频链接
  ↓
逐一打开视频 → 观看 20 秒 → 点赞
  ↓
打印统计结果，结束
```

---

## 配置参数（config.py）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `VIDEO_COUNT` | `20` | 每次运行点赞的视频数量 |
| `WATCH_DURATION` | `20` | 每个视频的观看时长（秒）|
| `MIN_WAIT / MAX_WAIT` | `1.0 / 3.0` | 操作间随机等待时间范围（秒）|
| `HEADLESS` | `False` | 是否隐藏浏览器窗口（建议保持 False）|
| `LOGIN_TIMEOUT` | `180` | 等待手动登录的超时时间（秒）|

修改 `config.py` 中的值即可调整脚本行为，无需改动其他文件。

---

## 注意事项

- 本脚本仅供学习交流，请勿用于批量刷量等违反平台规定的行为
- 每次运行都需要手动登录，不会保存账号信息
- 如果 B站更新页面结构导致选择器失效，可在 `liker.py` 的 `_LIKE_SELECTORS` 列表中添加新选择器
